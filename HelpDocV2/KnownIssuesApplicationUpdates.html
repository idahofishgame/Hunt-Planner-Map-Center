Known Issues & Application Updates

INTRODUCTION: Known Issues & Application Updates


Updates

  • 3/16/2018 Base Code Update & Revamped Layer List. Base code (jquery, bootstrap, arcgis js) running the application was brought up-to-date. A new layer list was created to better handle use of single services.
  • 8/18/2017 Significant Closure Areas Added. A large portion of private Potlatch land was purchased by Wilks Development in central Idaho in the recent past.  There are now allowing only limited access to the public.  Please contact them at info@wilksdevelopment.com to inquire about access. The Boise National Forest has implemented a closure associated with a timber salvage project in the area of last years Pioneer Fire in central Idaho. More information.
  • 7/14/2017 New Idaho Department of Lands Endowment Lands Layer Added. Produced by IDL.  Shows lands designated as Yes or No public access. Yes: these lands can be accessed by foot or water.  Some lands may have motorized access.  No: these lands are not accessible because they are surrounded by private landowners or because activities occurring on the lands make them unsafe for the public, such as within a military training area.
  • 6/20/2017 New Land Cover Layer Added. Produced by the Multi-Resolution Land Characteristics Consortium.  For more information about this layer please visit the 'Turn Layers On/Off' section of the help documentation.
  • 6/16/2017 New Public Road Closure Layer Added. All Wildfire-related Layers turned on.  Idaho Department of Parks and Recreation is how maintaining a layer depicting public (USFS, BLM, etc.) road closures.  It was added as a layer to the Map Center.  As a few wildfires have already been reported, all wildfire layers have been re-initialized.
  • 4/28/2017 Removed Google Basemaps and added Esri high-resolution Vector Basemaps.
  • 3/17/2017 Map Center Code Update. The Map Center base code was updated and a few changes were made to the customer interface.  The biggest change is a new Table of Contents (Turn Layers On/Off) widget. The tool sidebar can now be toggled open/closed on all platforms (desktop, tablet, and mobile).Because of the code update,  the NAIP Imagery basemap option was removed.
  • 7/22/2016 2015 Idaho NAIP Imagery Basemap Added.  One-meter resolution National Agriculture Imagery Program (NAIP) imagery was added as a basemap option.
  • 7/1/2016 Re-activated Wildfire Spatial Data Layers.  Fire emergency area closures, active fire perimeters, MODIS fire detection, and past years' fire perimeter data layers are now available as viewable layers.
  • 6/27/2016 Updated Waterfowl Hunt Areas for the 2016-17 season.  Includes ducks, geese, coots, snipes, and sandhill cranes.
  • 2/17/2016 PDF Maps Created Using the Create Printable Map tool are now Georeferenced. You can use the Avenza PDF Maps app to interact with a map to view your location, record GPS tracks, add placemarks, and find places on your Apple or Android device. You can also measure height, width, or area and view cursor coordinates using the measure tools in Adobe Acrobat Reader. The maps use coordinate system WGS 84.
  • 1/29/2016 Updated Turkey Controlled Hunts. Upland Game, Furbearer, and Turkey seasons and rules have been updated for 2016-17.  The turkey controlled hunts are now up-to-date.
  • 12/16/2015 Updated to Esri JavaScript to 3.10.
  • 12/16/2015 Added Popups to the Hunting Restrictions Layers.  Now when you turn on one of the hunting restrictions layers (ie. big game, furbearer, upland game bird, waterfowl) and click an area, an information popup will appear that gives information about the restriction or closure.
  • 11/16/2015  All Emergency Fire Closures Lifted.  All fire emergency area closures have been lifted.  All wildfire spatial data layers have been disabled until next fire season.  Hunt Restriction Alert.  If a hunt area has further restrictions inside the displayed boundary,  an alert will now popup telling the hunter just that.  Upon closing the popup, the webpage for IDFG hunting regulations will automatically open in a new tab so that hunters can review the restrictions.
  • 9/22/2015 Private Timberland Fire Closure Lifted. All major industrial timberlands are open to public access, subject to their normal terms and conditions.
  • 9/17/2015 Updated Waterfowl Hunt Areas.  Updated the 'Highlight a Hunt Area' waterfowl hunt areas list for the 2015 season.
  • 9/11/2015 Private Timberland Fire Closure Reduced. Some private timberlands are still closed to the public while others are open to foot traffic only.
  • 8/31/2015 'Create Printable Map' Tool Disabled.  Due to high server load we have temporarily disabled printing to keep the Map Center available.
  • 8/27/2015 Added Private Timberlands Fire Closures Layer. Private timberland companies in Idaho have closed there lands due to fire danger. More Info
  • 8/13/2015 Added NAD 27 UTM Zone 11 & 12 Coordinates.   As users are scrolling around the map, they will now see NAD27 UTM Zone 11N or 12N coordinates along with decimal degree coordinates.  The display has been moved from the lower right-hand corner to the upper right-hand corner.
  • 7/15/2015 Added Control Hunt Numbers. Users can now see the controlled hunt numbers when highlighting a hunt area using the 'Controlled Hunts:' dropdown menuu.
  • 7/7/2015 Re-activated Wildfire Spatial Data Layers.  Fire emergency area closures, active fire perimeters, and MODIS fire detection data layers are now available as viewable layers.
  • 3/30/2015 Added USFS Legislative Boundary Layer.  Some hunt area descriptions refer to the USFS legislative boundary.  It was added to the layer list for sportsmen to reference.  Links to full metadata were added to the 'Turn Layers On/Off' section of the help documentation.
  • 3/27/15 Different Symbology for Flagged Hunt Areas. Hunt areas with further restrictions, such as on private land only, are now symbolized with black diagonal lines.  A link to the IDFG hunting regulations web page has been added under the 'Highlighted Hunt Areas' legend label.
  • 3/25/15 Updated Deer and Elk Controlled Hunt Areas. Deer and elk controlled hunt areas have been updated for the 2015-2016 season.
  • 2/18/15 Updated Moose, Sheep, and Goat Controlled Hunt Areas.  Moose, sheep, and goat controlled hunt areas have been updated for the 2015-2016 season.
  • 11/10/14 Updated Wildfire Related Layers. There are no active wildfires or area closures in Idaho at this time.  The fire area closures and fire perimeters data layers have been removed from the 'Wildfire & Closure Related Layers" layer list until next fire season.  The MODIS Fire Detections layer will remain in the layer list.
  • 11/7/2014 Highlight a Hunt Area Improvements. Now when you highlight a hunt area, a label will also appear on the map.  Download Highlighted Hunt Area(s) as Google Earth kml(s) tool added. After a user highlights hunt areas using the 'Highlight a Hunt Area' tool, a new tool will appear called 'Download Highlighted Area as KML'.  When the user opens this window, they will find a hyperlink that will open/download the KML. Upload Waypoints & Tracks tool added.  Users can simply a GPS data such as waypoints and tracks (or other coordinate based datasets) to the map using this tool.  
  • 10/27/2014 Layer List/Legend Improvements.  The 'Turn Layers On/Off' option now opens in a collapsible window in the left-hand sidebar instead of in a modal window.  This allows the Map Center users to view the legend while looking at the map.  'Share the Map View' Tool added.  If a user would like to share a specific map view, they can copy the generated URL.  The URL will query for a center coordinate and map zoom level.  General Game Distribution Added.  Users can now add the General Distribution of different game species using a dropdown menu in the 'Highlight a Hunt Area' window.
  • 9/17/2014 Print Tool Improvements. Users now have the option to create PDF or PNG maps of 4 different formats: 8x11 Landscape, 11x17 Landscape, 8x11 Portrait, or 11x17 Portrait. The print DPI was also increased from 96 to 300.  This improves the print quality of the vector layers (hunt boundaries, etc.) that overlay the basemaps.
  • 9/5/2014 Highlight Multiple Hunt Areas. Improved the 'Highlight a Hunt' tool so that users can select multiple hunt areas from one category (ex. GMU, elk zones) or multiple hunts from different hunt areas.  Hunt areas from different categories will be highlighted in different colors.
  • 9/5/2014 Swapped GeoMAC for USFS-RSAC MODIS layer. The USFS-RSAC WMS layer has nicer symbology than the GeoMAC feature layer.
  • 9/5/2014 Added Data Source Links. There are a number of layers in the layers list that are maintained by different agencies.  For each, we now provide a link to metadata or a website where the data came from. (Land Mgmt Layer, Roads & Trails, etc.)
  • 8/25/2014 Removed the Motorized and Non-motorized Trails layer. The roads and trails layer has been, hopefully, temporarily removed.  It was brought to our attention that there is some road/trail information that was portrayed incorrectly by this layer.  As it is not maintained by IDFG,  we thought it was in the best interest of the public to remove the layer for the time being.  RESOLVED.  The trails layer is available.
  • 8/25/2014 Added 2014-2015 Waterfowl Hunt Areas
  • 7/31/2014 Open PDF from 'Create PDF Map' window. Now once a PDF is generated using the 'Create PDF Map' tool, a hyperlink title "CLICK HERE TO DOWNLOAD PDF MAP" will appear in the window.  When the user clicks the hyperlink, their PDF will open in a new browser window. (Work around for PDFs getting blocked by Popup Blockers).
  • 7/24/2014 Added Google Basemaps. After receiving feedback about the new Esri basemap options, the Google basemaps (roads, imagery, hybrid, and terrain) were also added to the basemap gallery.


Known Issues

  • 7/9/2014 Clear Measurement Graphics Before Attempting to Create a PDF - All measurement related graphics must be cleared before you attempt to create a PDF map.  Please press the 'Clear Measurement Results' button in the 'Measurements & Coordinates' window before attempting to create a PDF. If you do not clear the graphics, the print tool will fail.  RESOLVED.
  • 7/1/2014 Created PDF Map Incorrect or Rendering an Older Version - Sometimes when a user creates a PDF, an older version will appear. SOLUTION: Refresh the PDF window (press the 'F5' button on your keyboard).  RESOLVED.
  • 7/1/2014 Map Legend Labels Missing for Graphics Layers - The legend label(s) for added graphics layers (from highlighting a hunt area or using the draw tools) will say 'Override 1', 'Override 2', etc.  No solution has been found at this time.
